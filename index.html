**Política de Privacidade – NIVELio** Última atualização: 2026 O aplicativo NIVELio foi desenvolvido com o objetivo de monitorar níveis de água, status de sistemas hidráulicos e auxiliar na gestão eficiente de recursos hídricos em condomínios e instalações. ## 1. Coleta de Dados O NIVELio pode coletar informações como: * Dados de uso do aplicativo * Informações técnicas do dispositivo * Dados enviados pelos dispositivos conectados (como níveis de reservatórios) O aplicativo não coleta dados pessoais sensíveis sem consentimento do usuário. ## 2. Uso das Informações As informações coletadas são utilizadas para: * Exibir dados de monitoramento em tempo real * Melhorar o desempenho e a experiência do aplicativo * Garantir o funcionamento adequado do sistema ## 3. Compartilhamento de Dados O NIVELio não vende, aluga ou compartilha dados pessoais com terceiros, exceto quando necessário para funcionamento do serviço (como infraestrutura de servidores). ## 4. Segurança Os dados são transmitidos utilizando protocolos seguros (HTTPS) e armazenados de forma protegida. ## 5. Permissões O aplicativo pode solicitar permissões como: * Acesso à internet * Bluetooth (quando necessário para comunicação com dispositivos) Essas permissões são utilizadas exclusivamente para o funcionamento do sistema. ## 6. Alterações Esta política pode ser atualizada periodicamente. Recomendamos a revisão regular. ## 7. Contato Para dúvidas ou solicitações: Email: [contato@hottzlab.com](mailto:contato@hottzlab.com)
